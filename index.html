The Mandelbrot Set

A fractal is a never-ending pattern that repeats itself at every level of magnification. Fractals can be found everywhere in nature, think of trees composed of branches that are composed of branches, and a small sand dune has the same shape as a huge sand dune.

Fascinated by the beauty in the chaos and irregularity of fractals, I created a visualization of the Mandelbrot set using a recursive algorithm.

The Julia Set

For each point on the complex plane, a different Julia set is generated. The Mandelbrot set acts like an index to the Julia set in the sense that every point in the Mandelbrot set spawns a connected Julia set while those outside of it spawn disconnected ones.

I created an interactive program to show how the Julia set looks depending on your mouse position on the canvas.
